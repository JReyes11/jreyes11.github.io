Google Chrome Extension (WeatherApp)

"WeatherApp": Google Chrome extension I developed that leverages multiple API's to reteive weather information based on user input.



Data Endpoints and Animations
Endpoints leveraged:
  • GET http://api.weatherapi.com/v1/
  • GET https://restcountries.com

Animations returned from local respository.

The intent of building this tool
I wanted to showcase the power and convenience of leveraging Chrome extensions for internal tool development. This can easily be shared and utilized cross-functionally among technical and non technical teammates alike.

Some of the more common use cases (from the quality perspective) typically include generating internal user accounts, generating mock data, retrieving log files and creating data files ( CSV, JSON, etc...)

Data validation scripts in Node.js


Working in a fast pace environment may require multiple rounds of regression and integration testing as new code is merged during the given sprint.

In an effort to provide instant feedback, leveraging Node.js to develop validation scripts helps expedite regression cycles, allowing for additional time to focus on other areas of the feature or application.

In this example, I developed a data validation script, allowing for user input, while asserting the expected results are returned in the request playload.
